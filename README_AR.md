# 🏥 نظام تشخيص أمراض القلب بالذكاء الاصطناعي

## 📋 نظرة عامة

مشروع متكامل لتشخيص أمراض القلب (خاصة العصيدة القلبية ونقص التروية) باستخدام:
- 🤖 نموذج Random Forest للتعلم الآلي
- 🔧 حساس AD8232 لقراءة إشارات ECG
- 🎛️ لوحة Arduino Uno للتحكم
- 🐍 برنامج Python للتنبؤ والعرض

---

## 📁 الملفات الرئيسية

### 1. **complete_system.py** ⭐ (البرنامج الرئيسي الموصى به)
```
البرنامج المدمج الذي يجمع كل شيء في مكان واحد:
✅ تدريب النموذج
✅ قراءة البيانات من الأردوينو
✅ التنبؤ بحالة القلب
✅ عرض النتائج
✅ حفظ النتائج
```

**كيفية الاستخدام:**
```bash
python3 complete_system.py
```

**الخيارات:**
```
1️⃣  تدريب النموذج (المرة الأولى فقط)
2️⃣  قراءة البيانات من الأردوينو والتنبؤ
3️⃣  عرض النتائج المحفوظة
4️⃣  خروج
```

---

### 2. **train_model.py** (تدريب النموذج)
```
كود منفصل لتدريب النموذج فقط
يُستخدم إذا أردت تدريب النموذج بشكل منفصل
```

**الاستخدام:**
```bash
python3 train_model.py
```

**الناتج:**
- `heart_disease_model.pkl` - النموذج المدرب

---

### 3. **arduino_ecg_sensor.ino** (كود الأردوينو)
```
كود الأردوينو الذي يقرأ من حساس AD8232
ويستخلص الميزات ويرسلها إلى الكمبيوتر
```

**الخطوات:**
1. افتح Arduino IDE
2. انسخ محتوى الملف
3. اختر اللوحة والمنفذ
4. اضغط Upload

---

### 4. **python_arduino_integration.py** (قراءة البيانات)
```
برنامج منفصل لقراءة البيانات من الأردوينو
والتنبؤ بحالة القلب
```

**الاستخدام:**
```bash
python3 python_arduino_integration.py
```

## 🚀 البدء السريع

### الخطوة 1: التثبيت
```bash
# تثبيت المكتبات المطلوبة
pip install pyserial scikit-learn numpy matplotlib seaborn scipy
```

### الخطوة 2: توصيل الأردوينو
```
AD8232 GND  → Arduino GND
AD8232 3.3V → Arduino 3.3V
AD8232 OUT  → Arduino A0
AD8232 LO-  → Arduino Pin 10
AD8232 LO+  → Arduino Pin 11
```

### الخطوة 3: رفع كود الأردوينو
1. افتح Arduino IDE
2. انسخ `arduino_ecg_sensor.ino`
3. اختر اللوحة والمنفذ
4. اضغط Upload

### الخطوة 4: تشغيل البرنامج
```bash
python3 complete_system.py
```

### الخطوة 5: اختر من القائمة
```
1 → تدريب النموذج (المرة الأولى)
2 → قراءة البيانات والتنبؤ
3 → عرض النتائج
4 → خروج
```

## 🎯 الميزات المستخلصة

النموذج يستخلص 10 ميزات من كل نبضة قلب:

| # | الميزة | الوصف |
| :--- | :--- | :--- |
| 1 | القيمة العظمى | أعلى نقطة في الإشارة |
| 2 | القيمة الصغرى | أقل نقطة في الإشارة |
| 3 | المتوسط | متوسط قيم الإشارة |
| 4 | الانحراف المعياري | تشتت البيانات |
| 5 | الطاقة | مجموع مربعات الإشارة |
| 6 | الانحراف (Skewness) | عدم التماثل |
| 7 | التفرطح (Kurtosis) | حدة الذروة |
| 8 | معدل التغيير | سرعة التغيير |
| 9 | متوسط ST ⭐ | **الأهم للكشف عن المرض** |
| 10 | تقاطعات الصفر | عدد تقاطعات الصفر |

## 🆘 استكشاف الأخطاء

### خطأ: "No module named 'serial'"
```bash
pip install pyserial
```

### خطأ: "heart_disease_model.pkl not found"
```
1. تأكد من تشغيل train_model.py أولاً
2. تأكد من أن الملف في نفس المجلد
```

### خطأ: "Could not open port COM3"
```
1. تأكد من توصيل الأردوينو
2. تأكد من المنفذ الصحيح
3. أغلق برنامج Arduino IDE
```
