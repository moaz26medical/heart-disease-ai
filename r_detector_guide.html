🔬 خوارزمية Pan-Tompkins

📖 مقدمة:

خوارزمية Pan-Tompkins هي واحدة من أشهر الخوارزميات لكشف نقاط R في إشارات ECG. تم تطويرها عام 1985 وتتميز بدقة عالية وسرعة معالجة ممتازة.

خطوات الخوارزمية:

📥 1. تصفية تمرير النطاق (Bandpass Filter)
↓
📈 2. الاشتقاق (Derivative)
↓
⚡ 3. التربيع (Squaring)
↓
🔄 4. تكامل النافذة المتحركة (Moving Window Integration)
↓
🎯 5. تحديد الحد الأدنى (Thresholding)
↓
✅ 6. تحسين النتائج (Refinement)

تفاصيل كل خطوة:

1️⃣ تصفية تمرير النطاق (Bandpass Filter)

الغرض: إزالة الضوضاء والتشويهات

النطاق: 0.5 - 50 Hz

الترتيب: 1

الصيغة: استخدام مرشح Butterworth

2️⃣ الاشتقاق (Derivative)

الغرض: تحديد معدل التغيير في الإشارة

الطريقة: استخدام np.gradient()

الفائدة: تعزيز الانتقالات الحادة (مثل R peak)

3️⃣ التربيع (Squaring)

الغرض: تضخيم القيم الكبيرة وتقليل الضوضاء

الصيغة: squared = diff²

الفائدة: جعل R peaks أكثر وضوحاً

4️⃣ تكامل النافذة المتحركة (Moving Window Integration)

الغرض: تجميع المعلومات على فترة زمنية

حجم النافذة: 150 ms (0.15 * fs)

الطريقة: استخدام np.convolve()

5️⃣ تحديد الحد الأدنى (Thresholding)

الحد: mean(integrated) × 1.2

الغرض: تحديد النقاط التي تتجاوز الحد

النتيجة: قائمة بالنقاط المرشحة

6️⃣ تحسين النتائج (Refinement)

فترة الحماية: 200 ms (0.2 * fs)

الغرض: تجنب كشف نفس الضربة مرتين

التحسين: البحث عن الحد الأقصى المحلي

📊 خطوات معالجة الملف

1️⃣ تحميل الملف:

record = wfdb.rdrecord(file_path)
fs = record.fs  # تردد العينات
p_signal = record.p_signal  # البيانات الخام

2️⃣ كشف نقاط R:

sig0 = record.p_signal[:, 0]
r_peaks = pan_tompkins_detect(sig0, fs)

3️⃣ تحميل التعليقات:

ann = wfdb.rdann(file_path, "atr")
ann_samples = ann.sample  # موقع التعليق
ann_symbols = ann.symbol  # نوع التعليق

4️⃣ تجزئة الضربات:

for r_idx in r_peaks:
    start = int(r_idx - 0.2 * fs)  # 200 ms قبل
    end = int(r_idx + 0.6 * fs)    # 600 ms بعد
    beat = signal[start:end]
    time = np.linspace(-0.2, 0.6, len(beat))

5️⃣ حفظ النتائج:

df = pd.DataFrame({
    "time (s)": time,
    f"{lead} (mV)": beat
})
df.to_csv(output_path, index=False)

معاملات المعالجة:

المعامل القيمة الوصف

Highcut 50 Hz الحد الأقصى للتصفية
Window Before -0.2 s الوقت قبل R peak
Window After 0.6 s الوقت بعد R peak
Refractory Period 200 ms الفترة بين الضربات
