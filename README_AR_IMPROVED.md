# 🏥 نظام تشخيص أمراض القلب بالذكاء الاصطناعي

## 📋 نظرة عامة

مشروع متكامل لتشخيص أمراض القلب (خاصة العصيدة القلبية ونقص التروية) باستخدام:
- 🤖 نموذج Random Forest للتعلم الآلي
- 🔧 حساس AD8232 لقراءة إشارات ECG
- 🎛️ لوحة Arduino Uno للتحكم
- 🐍 برنامج Python للتنبؤ والعرض

---

## 🎯 الهدف من المشروع

تطوير نظام ذكي يستطيع:
1. ✅ قراءة إشارات القلب الكهربائية (ECG)
2. ✅ معالجة البيانات واستخلاص الميزات المهمة
3. ✅ التنبؤ بوجود أمراض القلب
4. ✅ عرض النتائج بشكل واضح وسهل الفهم
5. ✅ حفظ النتائج للمتابعة

---

## 📁 الملفات الرئيسية

### ⭐ البرنامج الرئيسي: `complete_system.py`

**الوصف:**
البرنامج المدمج الذي يجمع كل شيء في مكان واحد

**المميزات:**
- ✅ تدريب النموذج
- ✅ قراءة البيانات من الأردوينو
- ✅ التنبؤ بحالة القلب
- ✅ عرض النتائج
- ✅ حفظ النتائج

**كيفية الاستخدام:**
```bash
python3 complete_system.py
```

**الخيارات:**
```
1️⃣  تدريب النموذج (المرة الأولى فقط)
2️⃣  قراءة البيانات من الأردوينو والتنبؤ
3️⃣  عرض النتائج المحفوظة
4️⃣  خروج
```

---

### 🐍 برنامج تدريب النموذج: `train_model.py`

**الوصف:**
كود منفصل لتدريب النموذج فقط

**الاستخدام:**
```bash
python3 train_model.py
```

**الناتج:**
- `heart_disease_model.pkl` - النموذج المدرب
- `model_results.png` - الرسوم البيانية

---

### 🔧 كود الأردوينو: `arduino_ecg_sensor.ino`

**الوصف:**
كود الأردوينو الذي يقرأ من حساس AD8232

**الخطوات:**
1. افتح Arduino IDE
2. انسخ محتوى الملف
3. اختر اللوحة: Tools → Board → Arduino Uno
4. اختر المنفذ: Tools → Port → COM3
5. اضغط Upload

---

### 🐍 برنامج قراءة البيانات: `python_arduino_integration.py`

**الوصف:**
برنامج منفصل لقراءة البيانات من الأردوينو والتنبؤ

**الاستخدام:**
```bash
python3 python_arduino_integration.py
```

## 🚀 البدء السريع

### الخطوة 1️⃣: التثبيت

```bash
# تثبيت المكتبات المطلوبة
pip install pyserial scikit-learn numpy matplotlib seaborn scipy
```

### الخطوة 2️⃣: توصيل الأردوينو

```
الاتصالات:
┌─────────────────────────────────────┐
│         AD8232 ← → Arduino          │
├─────────────────────────────────────┤
│ GND      ← → GND                    │
│ 3.3V     ← → 3.3V                   │
│ OUT      ← → A0                     │
│ LO-      ← → Pin 10                 │
│ LO+      ← → Pin 11                 │
└─────────────────────────────────────┘
```

### الخطوة 3️⃣: رفع كود الأردوينو

1. افتح Arduino IDE
2. انسخ `arduino_ecg_sensor.ino`
3. اختر اللوحة والمنفذ
4. اضغط Upload

### الخطوة 4️⃣: تشغيل البرنامج

```bash
python3 complete_system.py
```

### الخطوة 5️⃣: اختر من القائمة

```
1 → تدريب النموذج (المرة الأولى)
2 → قراءة البيانات والتنبؤ
3 → عرض النتائج
4 → خروج
```

## 🆘 استكشاف الأخطاء

### ❌ خطأ: "No module named 'serial'"
```bash
# الحل:
pip install pyserial
```

### ❌ خطأ: "heart_disease_model.pkl not found"
```
الحل:
1. تأكد من تشغيل train_model.py أولاً
2. تأكد من أن الملف في نفس المجلد
3. أو عدّل المسار في البرنامج
```

### ❌ خطأ: "Could not open port COM3"
```
الحل:
1. تأكد من توصيل الأردوينو بـ USB
2. تأكد من المنفذ الصحيح
3. أغلق برنامج Arduino IDE
4. أعد تشغيل البرنامج
```
