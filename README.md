# 🏥 مشروع السنة الخامسة هندسة طبية
## نظام تشخيص أمراض القلب بالذكاء الاصطناعي

### 📋 نظرة عامة
هذا المشروع يهدف إلى بناء نظام متكامل لتشخيص أمراض القلب (خاصة العصيدة القلبية) باستخدام الذكاء الاصطناعي.

### 🎯 الأهداف
1. اقتباس إشارات القلب الحقيقية من جسم الإنسان
2. معالجة الإشارات الرقمية وتنظيفها
3. تدريب نموذج ذكاء اصطناعي على البيانات الطبية الحقيقية
4. تشخيص أمراض القلب تلقائياً

### ⚙️ المكونات العتادية
- **حساس AD8232:** لاقتباس إشارات القلب
- **Arduino:** معالج ميكروي للتحويل الرقمي
- **أقطاب ECG:** لالتقاط الإشارة من الجسم

### 💻 المكونات البرمجية
- **Python 3.11:** معالجة البيانات والذكاء الاصطناعي
- **scikit-learn:** نماذج التعلم الآلي
- **TensorFlow/Keras:** الشبكات العصبية
- **wfdb:** قراءة قواعس البيانات الطبية

### 📊 قواعس البيانات
- MIT-BIH Arrhythmia Database
- PTB Diagnostic ECG Database
- Long-Term ST Database
- UCI Heart Disease Dataset

### 🚀 الخطوات التالية
1. تحميل البيانات من PhysioNet
2. تدريب نموذج الذكاء الاصطناعي
3. ربط الحساس مع الأردوينو
4. اختبار النظام الكامل
