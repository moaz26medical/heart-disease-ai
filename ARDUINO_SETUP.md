# 🏥 دليل توصيل حساس AD8232 مع الأردوينو

## 📋 المتطلبات

### المكونات الإلكترونية:
- **Arduino Uno** أو **Arduino Mega** أو **Arduino Nano**
- **حساس AD8232 ECG** (Heart Rate Sensor Module)
- **أقطاب كهربائية** (Electrodes) - 3 أقطاب
- **كابل USB** لتوصيل الأردوينو بالكمبيوتر
- **أسلاك توصيل** (Jumper Wires)

---

## 🔌 خطوات التوصيل

### الخطوة 1: توصيل الحساس بالأردوينو

```
AD8232 Sensor          Arduino
─────────────────────────────
GND        ────────→  GND
3.3V       ────────→  3.3V (أو 5V مع مقاوم)
OUT        ────────→  A0 (Analog Pin)
LO-        ────────→  Pin 10
LO+        ────────→  Pin 11
```

### الخطوة 2: توصيل الأقطاب الكهربائية

```
الأقطاب الثلاثة:
├─ الأحمر (RA)   → الذراع اليمنى
├─ الأسود (LA)  → الذراع اليسرى
└─ الأبيض (RL)  → الرجل اليمنى (مرجع أرضي)
```

### الخطوة 3: تنظيف الجلد

- نظف المنطقة بكحول أو ماء صابون
- اترك الجلد يجف تماماً
- هذا يحسن التلامس الكهربائي

### الخطوة 4: وضع الأقطاب

```
الموضع الصحيح:
┌─────────────────────────┐
│                         │
│  الأحمر (RA)            │
│  على الذراع اليمنى     │
│                         │
│  الأسود (LA)            │
│  على الذراع اليسرى     │
│                         │
│  الأبيض (RL)            │
│  على الرجل اليمنى      │
│                         │
└─────────────────────────┘
```

## 💻 تثبيت البرنامج

### الخطوة 1: تثبيت Arduino IDE

1. اذهب إلى: https://www.arduino.cc/en/software
2. حمل **Arduino IDE** الإصدار الأحدث
3. ثبت البرنامج

### الخطوة 2: فتح الكود

1. افتح Arduino IDE
2. انسخ محتوى `arduino_ecg_sensor.ino`
3. الصق الكود في IDE

### الخطوة 3: اختيار اللوحة والمنفذ

```
Tools → Board → Arduino Uno (أو اللوحة المستخدمة)
Tools → Port → COM3 (أو المنفذ المتاح)
```

### الخطوة 4: رفع الكود

1. اضغط على زر **Upload** (السهم الأيمن)
2. انتظر حتى ينتهي الرفع
3. ستظهر رسالة "Done uploading"

## ⚠️ استكشاف الأخطاء

### المشكلة: لا تظهر أي بيانات

**الحل:**
- تحقق من توصيل USB
- تأكد من اختيار المنفذ الصحيح
- جرب منفذ USB مختلف

### المشكلة: تحذير "فقدان الاتصال"

```
⚠️ تحذير: فقدان الاتصال! تحقق من توصيل الحساس
```

**الحل:**
- تحقق من توصيل الأقطاب الكهربائية
- تأكد من نظافة الجلد
- أعد وضع الأقطاب

### المشكلة: قيم غريبة جداً (0 أو 1023)

**الحل:**
- تحقق من توصيل الحساس بـ A0
- جرب توصيل مختلف
- تحقق من جودة الأسلاك

### المشكلة: الأردوينو لا يستجيب

**الحل:**
- أعد تشغيل الأردوينو (افصل وصل USB)
- حاول رفع الكود مرة أخرى
- جرب لوحة أردوينو مختلفة

3. **استخدام النموذج المدرب** للتنبؤ بالمرض
4. **عرض النتائج** على الشاشة أو الموقع

## 📝 معلومات تقنية

### معدل البود (Baud Rate)
- **115200**: سريع جداً (مستخدم حالياً)
- **9600**: بطيء لكن موثوق
- **57600**: وسط

### تردد العينات (Sampling Rate)
- **360 Hz**: كافٍ لـ ECG (مستخدم حالياً)
- **250 Hz**: يوفر موارد الأردوينو
- **500 Hz**: أكثر دقة لكن يستهلك موارد أكثر

### حجم المخزن المؤقت (Buffer Size)
- **288 عينة**: تمثل 0.8 ثانية (نبضة قلب واحدة)
- يمكن تغييره حسب الحاجة

## 🎯 نصائح مهمة

✅ **افعل:**
- نظف الجلد قبل وضع الأقطاب
- استخدم أقطاب جديدة إذا كانت القديمة تالفة
- تأكد من ثبات الأقطاب على الجلد
- اجلس بهدوء أثناء القراءة

❌ **لا تفعل:**
- لا تلمس الأقطاب أثناء القراءة
- لا تحرك الذراعين كثيراً
- لا تستخدم أقطاب رطبة جداً
- لا تشد الأقطاب بقوة على الجلد
