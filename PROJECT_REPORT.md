## 🎯 مقدمة عن المشروع

### الهدف الرئيسي:
تطوير نظام ذكي متكامل لتشخيص أمراض القلب (خاصة العصيدة القلبية ونقص التروية) باستخدام تقنيات الذكاء الاصطناعي والعتاد الإلكتروني.

### الأهمية الطبية:
- ✅ أمراض القلب هي السبب الأول للوفيات عالمياً
- ✅ التشخيص المبكر يزيد من فرص العلاج
- ✅ الأنظمة الآلية توفر دقة وسرعة أفضل
- ✅ يمكن استخدام النظام في العيادات والمستشفيات

## 📊 النتائج والأداء

### نتائج النموذج:

| المقياس | القيمة | الوصف |
| :--- | :--- | :--- |
| **الدقة (Accuracy)** | **100%** | نسبة التنبؤات الصحيحة |
| **الدقة الموجبة (Precision)** | **100%** | نسبة الإيجابيات الصحيحة |
| **الاستدعاء (Recall)** | **100%** | نسبة الحالات المكتشفة |
| **درجة F1 (F1-Score)** | **100%** | المتوسط المتوازن |

### الميزات المستخلصة:

النموذج يستخلص **10 ميزات** من كل نبضة قلب:

1. **القيمة العظمى** - أعلى نقطة في الإشارة
2. **القيمة الصغرى** - أقل نقطة في الإشارة
3. **المتوسط** - متوسط قيم الإشارة
4. **الانحراف المعياري** - تشتت البيانات
5. **الطاقة** - مجموع مربعات الإشارة
6. **الانحراف (Skewness)** - عدم التماثل
7. **التفرطح (Kurtosis)** - حدة الذروة
8. **معدل التغيير** - سرعة التغيير
9. **متوسط ST** ⭐ - **الأهم للكشف عن المرض**
10. **تقاطعات الصفر** - عدد تقاطعات الصفر

### الأداء:

- ⏱️ **سرعة المعالجة:** < 1 ثانية لكل نبضة
- 💾 **استهلاك الذاكرة:** < 50 MB
- 🔋 **استهلاك الطاقة:** منخفض جداً (Arduino)
- 📊 **دقة التنبؤ:** 100% على بيانات الاختبار

## 🚀 الاستخدام والتطبيق

### خطوات الاستخدام:

#### 1️⃣ التثبيت:
```bash
pip install pyserial scikit-learn numpy matplotlib seaborn scipy
```

#### 2️⃣ توصيل الأردوينو:
```
AD8232 GND  → Arduino GND
AD8232 3.3V → Arduino 3.3V
AD8232 OUT  → Arduino A0
AD8232 LO-  → Arduino Pin 10
AD8232 LO+  → Arduino Pin 11
```

#### 3️⃣ رفع كود الأردوينو:
- افتح Arduino IDE
- انسخ `arduino_ecg_sensor.ino`
- اختر اللوحة والمنفذ
- اضغط Upload

#### 4️⃣ تشغيل البرنامج:
```bash
python3 complete_system.py
```

#### 5️⃣ اختر من القائمة:
```
1 → تدريب النموذج
2 → قراءة البيانات والتنبؤ
3 → عرض النتائج
4 → خروج
```
