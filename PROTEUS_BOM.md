## 🔌 تفاصيل التوصيلات

### توصيلات Arduino Uno

```
الدبابيس الرقمية:
├─ D0: RX (Serial)
├─ D1: TX (Serial)
├─ D2: LO+ من AD8232
├─ D3: LO- من AD8232
├─ D4: SDN من AD8232
├─ D5: LED الأحمر
├─ D6: LED الأخضر
├─ D7: متاح
├─ D8-D13: متاح

الدبابيس التناظرية:
├─ A0: ECG Signal من AD8232
├─ A1: متاح
├─ A2: متاح
├─ A3: متاح
├─ A4: SDA (I2C)
├─ A5: SCL (I2C)

الطاقة والأرضي:
├─ 5V: مصدر الطاقة 5V
├─ 3.3V: مصدر الطاقة 3.3V
├─ GND: الأرضي
```

### توصيلات AD8232

```
الدبابيس:
├─ 1: GND (أرضي)
├─ 2: 3.3V (طاقة)
├─ 3: OUTPUT → Arduino A0
├─ 4: LO+ → Arduino D2
├─ 5: LO- → Arduino D3
├─ 6: SDN → Arduino D4
├─ 7: RA (اليد اليمنى)
├─ 8: LA (اليد اليسرى)
└─ 9: LL (الرجل اليسرى)
```

### توصيلات ADS1115

```
العنوان I2C: 0x48

الدبابيس:
├─ 1: GND (أرضي)
├─ 2: VDD (3.3V)
├─ 3: SCL → Arduino A5
├─ 4: SDA → Arduino A4
├─ 5: ADDR (GND للعنوان الافتراضي)
├─ 6: ALRT (متاح)
├─ 7: A0 → AD8232 OUTPUT
├─ 8: A1 (متاح)
├─ 9: A2 (متاح)
└─ 10: A3 (متاح)
```

### توصيلات OLED Display

```
العنوان I2C: 0x3C

الدبابيس:
├─ 1: GND (أرضي)
├─ 2: VDD (3.3V)
├─ 3: SCL → Arduino A5
└─ 4: SDA → Arduino A4
```
