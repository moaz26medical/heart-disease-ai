📋 نظرة عامة على البرنامج

🎯 الهدف:

برنامج متكامل لقراءة إشارات ECG من Raspberry Pi في الوقت الفعلي وتحليلها باستخدام نموذج التعلم العميق

المميزات الرئيسية:

  • ✅ قراءة فعلية: قراءة إشارات ECG بتردد 250 Hz
  • ✅ عرض مباشر: رسم الإشارات على الشاشة في الوقت الفعلي
  • ✅ تنبؤ ذكي: استخدام نموذج TFLite للتنبؤ السريع
  • ✅ عرض OLED: عرض النتائج على شاشة OLED صغيرة
  • ✅ إحصائيات: حساب عدد كل نوع من الحالات
  • ✅ واجهة رسومية: عرض جميل للبيانات

المدخلات والمخرجات:

المعيار القيمة الوصف
تردد العينات 250 Hz عدد العينات المأخوذة في الثانية
حجم النافذة 200 عينة عدد العينات المستخدمة للتنبؤ
عدد الفئات 3 فئات Normal, Ischemia, Undiagnosed
حد الثقة 0.6 الحد الأدنى لقبول النتيجة
الإشارات 3 إشارات Lead II, Lead III, aVF

🔧 المكونات والأجهزة

🍓 Raspberry Pi 4 Model B

المعالج: ARM Cortex-A72 رباعي النوى

الذاكرة: 4GB RAM

الوصلات: USB, GPIO, I2C, SPI

نظام التشغيل: Raspberry Pi OS (Linux)

⚡ ADS1115 - محول رقمي

الدقة: 16 بت

الوصلة: I2C

العناوين: 0x48 - 0x4B

الوظيفة: تحويل الإشارات التناظرية إلى رقمية

📊 شاشة OLED SH1106

الحجم: 128x64 بكسل

الوصلة: I2C (العنوان: 0x3C)

الوظيفة: عرض النتائج والإحصائيات

🎯 حساس AD8232

الوظيفة: قراءة إشارات ECG

الحساسية: عالية جداً

الضوضاء: منخفضة جداً

توصيل الأجهزة:

الجهاز الوصلة Raspberry Pi الملاحظات
ADS1115 I2C GPIO 2, 3 العنوان: 0x48
OLED SH1106 I2C GPIO 2, 3 العنوان: 0x3C
AD8232 Analog ADS1115 (P0, P1) Lead II, Lead III

4️⃣ دالة التحديث الرئيسية:

def update():
    # قراءة الإشارات
    vII = leadII_channel.voltage
    vIII = leadIII_channel.voltage
    vAVF = (vII + vIII) / 2
    
    # تحديث البيانات
    data_leadII = np.roll(data_leadII, -1)
    data_leadII[-1] = vII
    
    # رسم على الشاشة
    curve_leadII.setData(time_data, data_leadII)
    
    # إضافة إلى المخزن المؤقت
    buffer_leadII.append(vII)
    
    # التنبؤ عند امتلاء المخزن
    if len(buffer_leadII) >= window_size:
        segment = np.array(buffer_leadII[-window_size:])
        segment = (segment - np.mean(segment)) / (np.std(segment) + 1e-8)
        
        # التنبؤ
        data = np.array(segment, dtype=input_details[0]["dtype"]).reshape(
            input_details[0]["shape"]
        )
        interpreter.set_tensor(input_details[0]["index"], data)
        interpreter.invoke()
        output_data = interpreter.get_tensor(output_details[0]["index"])[0]
        
        # الحصول على النتيجة
        max_prob_index = np.argmax(output_data)
        max_prob = output_data[max_prob_index]
        
        result = classes[max_prob_index] if max_prob >= threshold else "Undiagnosed"
        counters[lead_name][result] += 1

sudo raspi-config
# اختر: Interfacing Options -> I2C -> Enable

3️⃣ التحقق من الأجهزة:

sudo i2cdetect -y 1
# يجب أن تظهر:
# 0x3C - OLED
# 0x48 - ADS1115
